1. Buka Laravel Tinker dengan perintah:
    php artisan tinker
2. menambahkan data ke model Blog
    App\Models\Blog::create([
        'judul-blog' => 'Judul Artikel 1',
        'penulis-blog' => 'Boy Wiliam',
        'isi-blog' => 'Lorem ipsum dolor sit amet consectetur, adipisicing elit. Ea provident, nobis consequuntur quasi eligendi quisquam placeat voluptate? Temporibus unde asperiores ducimus maxime ipsa amet optio, ex, veniam consequatur molestias vero.'
    ]);

3. menampilkan semua data di model Blog
    App\Models\Blog::all()

4. menampung ke variabel:
    $blogs= App\Models\Blog::all()

5. manampilkan data pertama:
    $blogs->first()

6. mencari data dgn id 4
    $blogs->find(4)

7. menghapus semua data
    $blogs->delete()
